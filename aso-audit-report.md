## 1. Title — 7/10

### Current (en-US)
```

```

### Analysis

| Check | Status | Notes |
|-------|--------|-------|
| Primary keyword present | ✅ | "File Transfer" — the #1 category term |
| Character usage | ✅ | 28/30 — near max |
| Brand vs keyword balance | ✅ | "Airclap" brand + keyword combo |
| Readability | ✅ | Natural, not stuffed |
| Uniqueness | ⚠️ | "Fast File Transfer" describes 50+ apps |

### Issues

1. **"Fast" is generic** — Every file transfer app claims to be fast. Not differentiating.
2. **Missing cross-platform signal** — The #1 differentiator (works across ALL devices) isn't in the title. Users searching "cross platform file transfer" won't see a strong match.

## 7. Ratings & Reviews — 3/10 ⚠️ CRITICAL

### Current State

| Metric | Value | Benchmark (Utilities) | Status |
|--------|-------|-----------------------|--------|
| Average Rating | 5.0 ★ | 4.5+ | ✅ Perfect |
| Rating Count | 8 | 100+ for visibility | ❌ **CRISIS** |
| Recent Reviews | Unknown | Positive trend | ❓ |
| Dev Responses | Unknown | Responds to all negative | ❓ |

### Why This Is Your #1 Problem

1. **Algorithm weight** — Apple's algorithm heavily factors rating count into keyword ranking. With 8 ratings, Airclap is essentially invisible in competitive keyword searches.
2. **Conversion killer** — Users see "5.0 (8)" and think "nobody uses this app." 50+ ratings is the minimum credibility threshold.
3. **Fragility** — One 1-star review drops you to ~4.4 stars. With 100+ ratings, a single bad review barely moves the needle.
4. **Competitor gap** — LocalSend likely has thousands of ratings. Even a better app loses if it looks unpopular.

### Impact Projection

| Rating Count | Expected Impact |
|--------------|-----------------|
| 8 → 50 | Keyword rankings begin improving, minimal credibility |
| 50 → 200 | Significant ranking boost, users trust the app |
| 200 → 500 | Competitive with mid-tier utilities |
| 500+ | Algorithm treats you as an established app |

### Immediate Action Required
Run `/rating-prompt-strategy` ASAP. This is blocking all other ASO improvements.
